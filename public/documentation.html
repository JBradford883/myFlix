myFlix Api Documentation

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoint Documentation

Description URL HTTP Method Query Parameters Request body data format Response body data format
Get list of all movies to the users /movies GET none none A JSON object holding data about all the movies
Get data about a single movie by title (description, genre, director, image URL, whether it’s featured or not) /movies/[title] GET /movies/:title none A JSON object holding data about a single movie containing the title, description, genre, director, image, and if it is featured

Example:
[
{ Title: The Dark Knight,
Description: Description about the movie,
Genre: Action,
Director: Christopher Nolan,
Image: thedarkknight.jpg,
Featured: true }
]
Get data about a genre (description) by name/title (e.g., "Thriller"). /movies/genre/[name] GET /movies/genre/:name none A JSON object holding data about the genre.
Get data about a director (bio, birth year, death year) by name. /movies/director/[name] GET /movies/director/:name none A JSON object holding data about the director.
Allow new users to register /users POST none A JSON object holding data about the user who registered.

Example:
{
Username:,
Password:,
Email:,
Birthday:,
}		 A JSON object holding data about the user who was added.

Example:
{
Username: Jbradford,
Password: 123456,
Email: email@address.com,
Birthday: 02-03-1988,
}
Allow users to update their user info (username). /users/[Username] PUT /users/:Username A JSON object holding data the user has updated.

Example:
{
Username:,
Password:,
Email:,
Birthday:,
}		 A JSON object holding data from the updated user information.

Example:
{
Username: Jbradford,
Password: 123456,
Email: email@address.com,
Birthday: 02-03-1988,
}
Allow users to add a movie to their list of favorites. /users/[Username]/favorites/[movieID] POST /user/:Username/favorites/:movieID none Shows a text message that the movie has been added to the users favorites list.
Allow users to remove a movie from their list of favorites. /users/[Username]/favorites/[MovieID] DELETE /user/:Username/favorites/:MovieID none Shows a text message that the movie has been removed from the users favorites list.
Allow existing users to deregister /users/[Username] DELETE /users/:Username none Shows a text message that the user has been removed.
